# Netflix-Crawling
Netflix review Crawling Python 업무자동화

1. 넷플릭스 주간 영화, TV Top10 기준으로 네이버 검색 후 크롤링

3. 컨텐츠 별 워드카운트 순위 top5가 포함된 excel과 워드클라우드를 이메일로 전송
